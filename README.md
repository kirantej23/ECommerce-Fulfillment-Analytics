# 📦 E-Commerce Supply Chain & Fulfillment Performance Analytics

## 🎯 Project Overview
An end-to-end data analytics project focused on diagnosing delivery delays and supply chain bottlenecks for a global e-commerce dataset of 100,000 orders. This project demonstrates skills across data engineering (Python data generation), database management (MySQL scripting), and visual business intelligence (Power BI dashboarding).

## 🛠️ Tech Stack & Workflow
1. **Data Engineering (Python):** Generated a robust dataset of 100,000 records modeling delivery windows, customer feedback, and regional segments.
2. **Database Management (MySQL):** Audited order lifecycle stages using `SELECT DISTINCT` to map unique fulfillment statuses and checked backend data integrity.
3. **Business Intelligence (Power BI):** Engineered DAX measures tracking Key Performance Indicators like OTDR % and Total Volume to deliver interactive executive insights.

## 💡 Key Business Insights Discovered
* **The Shipping Bottleneck:** Advanced SQL filtering proved that the **Standard** shipping method in the **North** region systematically breached our delivery targets.
* **Fulfillment Impact on Sentiment:** Visual chart analysis confirmed a direct negative correlation—customer star ratings drop heavily as delivery times exceed 4 days.
